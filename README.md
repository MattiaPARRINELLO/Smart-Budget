## 🎯 Présentation
Plateforme de contenu moderne construite avec Next.js 14, destinée à un public jeune. Articles optimisés SEO, newsletter intégrée, CMS headless, et architecture full-stack avec Prisma et NextAuth.

---

## ✨ Fonctionnalités
- ✅ Architecture full-stack Next.js 14 (App Router)
- ✅ CMS headless avec articles, catégories, tags
- ✅ Newsletter avec abonnements
- ✅ SEO optimisé (sitemap, metadata, OG images)
- ✅ Authentification NextAuth
- ✅ Base de données SQLite via Prisma

---

## 🚀 Installation
```bash
git clone https://github.com/MattiaPARRINELLO/Smart-Budget.git
cd Smart-Budget
npm install
cp .env.example .env
npx prisma db push
npm run dev
```
